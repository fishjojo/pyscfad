Getting started
===============

Installation
------------

The easiest way to install pyscfad is to
install it via pip from `PyPI <https://pypi.org/project/pyscfad/>`_.

.. code::

   pip install pyscfad

More sophisticated instructions for installation can be found in the :ref:`install` page.
